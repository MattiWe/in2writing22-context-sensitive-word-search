# in2writing22-neural-netspeak

## Quick Start

1. Setup and activate a venv and install the dependencies 

    ```bash
    ~$ python3 -m venv venv 
    ~$ . venv/bin/activate
    (venv) ~$ pip install wheel
    (venv) ~$ pip install .
    (venv) ~$ python -m spacy download en_core_web_trf
   ```

2. Start the cli and follow the help

   ```bash
   (venv) ~$ main
   ```
